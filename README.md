# 🍽️ Dish-Dash (Frontend)

Dish-Dash is a modern full-stack recipe sharing platform where users can discover delicious recipes, save their favorites, and share their own creations. It also features **Dish-Dash Assistant**, an AI-powered chatbot built with the **Gemini API** that helps users navigate the platform and answers questions about how to use it.

## 🌐 Live Demo

**Live Website:** https://dish-dash-mu.vercel.app

---

## ✨ Features

- 🍳 Browse recipes from the community
- 🔍 Search recipes by name
- 📊 Sort recipes for easier discovery
- ❤️ Save your favorite recipes
- ➕ Add your own recipes
- ✏️ Update recipes you've created
- 🗑️ Delete your own recipes
- 🤖 **Dish-Dash Assistant** powered by Gemini API
  - Explains how Dish-Dash works
  - Guides users through the platform
  - Answers questions about using the website
- 📱 Fully responsive design
- 🎨 Modern and clean user interface

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

- Node.js (v18 or later)
- npm, yarn, pnpm, or bun

### Installation

Clone the repository:

```bash
git clone https://github.com/Msayeem/Dish-Dash.git
```

Navigate to the project directory:

```bash
cd dish-dash-frontend
```

Install dependencies:

```bash
npm install
```

Create a `.env.local` file in the root directory:

```env
NEXT_PUBLIC_API_URL=YOUR_BACKEND_URL
NEXT_PUBLIC_BETTER_AUTH_URL=YOUR_AUTH_URL
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```
